# JavaMetodos

Este repositório contém exercícios em Java focado em métodos.

## Exercícios

- **Potência** – calcula uma potência e mostra fórmula e resultado  
- **Soma de Números** – soma de 1 até um número e mostra fórmula + resultado  
- **Tabuada** – gera tabuada de 1 a 10  
- **Par ou Ímpar** – decide par ou ímpar  
- **Média e Situação** – calcula média e retorna situação do aluno
